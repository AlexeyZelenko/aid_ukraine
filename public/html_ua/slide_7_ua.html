ПриватБанк

Інтеграція з API ПриватБанку для миттєвих переказів з будь-якої картки.

Можливості:

  • Автоматичні регулярні платежі
  • Платежі за QR-кодом
  • Перекази на рахунок фонду

МоноБанк

Інтеграція з МоноБанком для зручних платежів через мобільний додаток.

Можливості:

  • Банка для збору коштів
  • Платежі через додаток
  • Відстеження руху коштів

Принципи прозорості платежів

1. Надходження коштів

Автоматична фіксація внесків з усіх джерел у системі

2. Розподіл за потребами

Направлення внесків за обраними категоріями та проектами

3. Документування витрат

Завантаження чеків, актів та фото доставки допомоги

4. Публічна звітність

Періодичні звіти та можливість перевірити статус кожного платежу

Кожен благодійник може відстежити, куди саме пішли його кошти, і отримати підтвердження про надання допомоги.
